## The one-line thesis

> Swift's site has **effectively zero keyword targeting** today — which means this is a **greenfield build, not an optimization project**. The platform migration is the moment to build a scalable, technically-clean architecture from the ground up. The competitive war for a low-authority local operator is **the Google Map Pack and Local Services Ads**, not organic backlink parity with national franchises — so the plan wins **home turf (Boone + Ames) first**, on the levers that actually convert: reviews, local pages, and emergency-intent paid.

## Diagnostic snapshot (measured)

| Signal | Finding |
|---|---|
| Homepage title tag | *"Swift Restoration LLC - This site is used to help customers look into our company"* — placeholder, zero keywords |
| Service pages | **None** (service cards link to `#`) |
| Location pages | **None** (8 counties are unlinked text) |
| Crawl | ~93 URLs, **mostly WordPress image-attachment pages**; 42 `noindex` (24.85%) |
| Templating | Gutenverse plugin → duplicate titles / H1s / meta across ~51 URLs |
| Authority | **Domain Rating 3.2** (Ahrefs) |
| Reputation | **5.0★ on 15 Google reviews** — a strong, under-leveraged asset |
| Paid | 2 generic Search ads; live `&amp;` encoding bug; **no LSA presence** |

## Methodology & data sources

This case study separates **measured data** from **estimates** — a deliberate rigor choice.

- **Measured:** live site crawl (page structure, titles, H1s, meta), Ahrefs Domain Rating (free endpoint), Google Business Profile review data, Google Ads Transparency Center, Screaming Frog issue export.
- **Estimated (clearly labelled):** keyword volume **bands** (High/Med/Low) derived from population + intent, since exact MSV/difficulty requires a paid Semrush/Ahrefs Site-Explorer tier not active during this phase.
- **To be baselined:** organic traffic, impressions, and conversion metrics — established via the analytics stack in [Section 17](17-Analytics-and-Tracking/README.md) once GA4, call tracking, and GBP reporting are live.
